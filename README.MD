# Анализатор зависимостей пакетов

Этот проект представляет собой инструмент командной строки на Python для анализа и визуализации графа зависимостей пакетов.

## Основные возможности

- **Построение графа зависимостей**: Скрипт рекурсивно обходит зависимости указанного пакета до заданной глубины.
- **Два режима работы**:
  - `online`: Получает информацию о зависимостях из удаленного Maven-репозитория.
  - `test`: Читает граф зависимостей из локального текстового файла для тестирования и отладки.
- **Фильтрация**: Позволяет исключать пакеты из графа по подстроке в имени.
- **Обнаружение циклических зависимостей**: Предотвращает бесконечный обход при наличии циклов в зависимостях.
- **Консольный вывод**: Отображает итоговый граф в виде древовидной структуры в консоли.

## Как использовать

Скрипт запускается из командной строки с указанием необходимых параметров.

### Параметры

- `--package`: Имя пакета для анализа.
- `--repo`: URL Maven-репозитория или путь к файлу с тестовым графом.
- `--version`: Версия пакета.
- `--depth`: Глубина анализа зависимостей (по умолчанию `1`).
- `--filter`: Строка для фильтрации пакетов.
- `--mode`: Режим работы (`online` или `test`).

### Пример запуска

**Online-режим:**
```bash
python3 main.py --package "com.lumidion:unistore_3" --repo "https://repo1.maven.org/maven2" --version "0.2.0" --depth 2
```

**Тестовый режим:**
```bash
python3 main.py --package "A" --repo "test_graph.txt" --mode "test" --version "1.0" --depth 8
```
